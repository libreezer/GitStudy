# 1.初级句型
## 1.1 基本句和补语
### 1.1.1五种基本句型

| 形式 | 代表 |
| :- | -- |
| S+V | 主语+谓语 |
| S+V+O | 主语+谓语+宾语 |
| S+V+C | 主语+谓语+补语(主语+系动词+表语) |
| S+V+O+O | 主语+谓语+宾语+宾语 |
| S+V+O+C | 主语+谓语+宾语+补语 |

```
Breeze died in world war II.
```
`Breeze`是主语。`died`是谓语，而后面的介词短语只是用来补充说明`died`，因此即使没有后面的介词短语这个句子也成立。
==像`die`这种可以独立发生的动作，不牵涉到其他的人或物就叫不及物动词。==

```
Breeze killed three soldiers.
```
==在这个句子中`kill`这种动作必须发生在另一个对象上，因此就叫做及物动词，它必须要有一个宾语来承受这个动作。==其中的soldiers就是宾语。

根据第二句来看，如果去除掉宾语句子就没有表达出完整的意思，不过可以看出`Breeze`杀了个什么东西。反之，如果句子缺少补语就会变得没有意义，因为叙述部分完全缺乏。

==在所有的英语动词中，只有解释为“是”的动词是空的，完全没有意义。==只有解释为“是”的动词没有叙述能力，只能扮演引导叙述部分的角色。
```
Breeze was a soldier.
```

### 1.1.2 不必翻译的动词:be动词
```
Breeze is handsome.
```
中文里，如果后面跟的是形容词，动词的“是”就会被丢掉。
例如上面的句子，如果说“布瑞兹是帅气的”就会很奇怪，说明“是”这个动词是空的，完全没有意义，因此只能==把主语和后面构成叙述的部分连接起来，所以它又叫做“连缀动词(Linking Verb)。”==在这类动词的后面的部分补足了句子使它获得完整的意思，称之为“补语(Complement)”。

```
That dress _looks_ pretty.
The dog _seems_ frindly.
```
把上面的句子的动词全部换成be动词，这些句子的意思和句型都没有太大的改变。

### 1.1.3 宾语补语的句型
主语补语的句型是用补语告诉读者主语是什么，中间用“是”连接起来。
==“主语+谓语+宾语+补语”的句型则是用补语告诉读者宾语是什么，中间暗含一个“是”的存在。==

```
I find the dress pretty.
They consider his demands reasonable.
```
拿第一个句子来看，dress和pretty之间没有“是”但是有暗示性的存在。如果加个be动词进去，就变成了主语补语的句子，也就是检验S+V+O+C句型最简便的方法：
> 把宾语和补语拿出来，中间加be动词，看看能不能改成S+V+C

### 1.1.4 补语的词类
补语的词类应该是名词和形容词最合理，因为主语或宾语都是名词，所以补语也可以是名词，经由“是”的连接来表达同等关系。
`Breeze is a soldier`中`a soldier`就是名词，由`is`来进行连接。
### 1.1.5 没有补语的be动词
be动词是最纯粹的linking verb，解释为“是”，后面要有补语才算完整。==如果be动词后面没有补语，这个be动词并不是当做连缀动词。这时候be动词并不解释为“是”，而是解释为“存在”，用在S+V句型中。==

```
I think;therefore I am. 我思故我在
```
其中的be动词就不能翻译为“是”，而要翻译成“存在”。
### 1.1.6 两个宾语的句型
```
Breeze's father(S) gave(V) him(O) a dog(O).
Breeze's father(S) called(V) him(O) a dog(C).
```

## 1.2 名词短语与冠词
### 1.2.1 名词短语
英语是一种拼音文字，用词尾的变化来表示单复数，==一些符号来配合表示改名词的范围，这种符号在语言学上称为“限定词(Determiners)”。==

```
a new book
many good students
```
这几个名词短语由三部分组成：
1. 第一个部分（a,many）就是限定词，这部分决定第三个部分的单复数。
2. 第二个部分（new,good）则是形容词，为第三部分进行修饰。
3. 第三个部分（book,students）就是名词本身
名词短语的这三个部分都可以省略，比如a book也是成立的，例如`Of these answers, this one is the best.`其中`this one`就代指了`the best answer`。
### 1.2.2 什么时候不需要限定词
如果名词本身不适合以“一个”来交代，则可以省略冠词：
```
Unmarried men are a rare species these days.
```
在短语`unmarried men`中men已经表示为复数形式，自然不能再填冠词。不可数名词也不能够加冠词。
### 1.2.3 专有名词与补语位置
人名和地名都是专有名词，也不适合用a(n)，因此可以不用限定词。

```
Breeze, head of the football team.
```
`head of the football team`就是`who was head of the football team`的省略。
### 1.2.4 定冠词the的用法
the可以被视为that或those的弱化形式（a或an是one的弱化形式），而that和those是指示形容词，有明确的指示功能，因此the也可以这么理解。

```
He should be home, I saw a light in his house.
Turn off the portal light.
```

### 1.2.5 定冠词与专有名词
专有名词定义：
> 只有一个对象存在的名词，既然只存在一个对象就没有“这个”，“那个”的区别，也就不能加定冠词。

如果说this book就说明有that book，为了指明就要使用the book。
==如果加了the就说明这个东西有两个以上，就不是专有名词了。==

```
the Pacific (Ocean)
the Atlantic (Ocean)
```
其中Ocean就是普通的名词，但是有Pacific进行修饰，再加上the就变成了专有名词，指的是“那个叫Pacific的海”。

## 1.3 动词时态
现代语法中时间和状态是分开处理的，==时间观念就是（过去，现在，将来），状态观念还包含主动和被动等。==
简单来说就是：把be动词当做动词，后面的分词视为形容词补语。
<mark style="background: #FFB86CA6;">简单式的动词可以清晰的交代此动作是发生于哪个时段。</mark>

### 过去类
| 名称 | 形式 |
| --- | ---  |
| 一般过去时 | did |
| 过去完成时 | had done |
| 过去进行时 | was/were doing |
| 过去将来时 | would doing |

### 现在类

| 名称 | 形式 |
|---|---|
| 一般现在时 | do/does |
| 现在进行时 | be doing |
| 现在完成时 | have/has done |
| 现在完成进行时 | have/has been doing |

### 将来类
| 名称 | 形式 |
| --- | --- |
| 一般将来时 | will do/be going to do |
| 将来完成时 | will have done |
| 将来进行时 | will be doing |
| 将来完成进行时 | will have been done |


## 1.4 语气
依照各种不同程度的“真，假”口吻，可以细分为四种语气：

|名称|作用|
|--|--|
|叙事语气|表示所说是真的|
|条件语气|表示真假不确定|
|假设语气|表示与事实相反|
|祈使语气|表示希望是真的|

### 1.4.1 叙事语气
英语中的一般语气都是叙事语气。
现在，过去的事情真假已经确定，因此可以用叙事语气。未来的事情不确定真假，因此未来式动词要加上will助动词，因为助动词带有不确定性。
### 1.4.2 条件语气
句子中加入语气助动词（must，should等）就有不确定的语气了，称为条件语气。
语气助动词有以下两点注意事项：
**1. 表达时间的功能不完整**
**2. 用完成式表达对过去的猜测**
### 1.4.3 假设语气
#### 现在时间
```
If I were you, I wouldn't do it.
```

假设语气是用动词的过去形态表示“非事实”。
#### 过去时间
```
If I had known earlier, I might have done something.
```
时间本就是过去的状态，因此就用过去完成时来表示虚拟语气。

#### 未来时间
```
If an asteriod should hit the earth, man could die out.
```
未来的事不确定，但是发生的可能性低可以用非事实语气来叙述。

虚拟语气的主要从句中都会有过去式的助动词存在。
