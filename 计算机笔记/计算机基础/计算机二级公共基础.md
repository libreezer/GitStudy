# 计算机的发展

1. 1946年诞生，名称为ENIAC(埃尼阿克)，由军用到民用
2. 计算机之父：冯·诺依曼
3. 冯·诺依曼计算机原理：
	1. 采用二进制进行数据存储
	2. 程序和数据存放在存储中，无需人工干涉
4. 发展历程：
	1. (46-57年)电子管：稳定性好，但功耗和体积大
	2. (58-64年)晶体管：体积小
	3. (65-70年)中小规模集成电路
	4. (70-至今)大型超大型集成电路：度越高，体积越小，功耗越低
5. 计算机的应用：
	1. 科学计算：轨道计算/天气计算/云计算
	2. 信息处理：<mark>OA办公系统</mark>
	3. 过程控制：工业机器制造
	4. 辅助功能：工作时的辅助，如Office
	5. 其他常见：聊天/购物/影视等
# 信息的表示和存储

### 数据单位

- 比特/位（bit）：<u>位是度量数据的最小单位</u>。在计算机技术中用二进制表示数据，1位数据只能表示0和1两种代码。
- 字节（Byte）：<u>字节是度量数据的最小<mark>存储单位</mark></u>。一个字节（Byte）由8位（bit）二进制数字组成。存储器容量通常以字节为单位（Byte,B）来描述。
- 字长：一次能处理的二进制数，越长越快，如：32位/64位/128位

### @网络拓扑结构
1. 星型拓扑：网络风险集中在中心节点上；
2. 环型拓扑：每一个节点都存在稳定风险；
3. 总线型拓扑：搭建容易，成本最低；
4. 网状拓扑：部署成本高，网络稳定性好；
5. 树型拓扑：可满足汇集信息的应用要求，成本低，易扩建推广。

### @数据结构
#### 栈
栈是限定在一端进行插入和删除的线性表。
特点：
1. 只能在栈顶进行插入和删除
2. 修改原则是<mark style="background: #FF5582A6;">“先进后出”或“后进先出”</mark>
#### 队列
队列是指允许在一端进行插入，而在另一端进行删除的线性表。
特点：
1. 只许在队尾插入，队头删除
2. 修改原则是<mark style="background: #FF5582A6;">“先进先出”或“后进后出”</mark>
#### 循环队列
是讲队列存储空间的最后一个位置绕道第一个位置，形成逻辑上的环状空间
#### 线性链表
线性表可以采用顺序存储和链式存储。线性表的顺序存储叫做顺序表，线性表的链式存储结构叫做线性链表。![[../../_Repertory/计算机二级公共基础_pic/d79dc873696fa64d5a41e21fe655d18fba11dfe0.jpg@1256w_592h_!web-article-pic.webp]]
特点：
1. 各数据结点的存储空间可以不连续；<mark style="background: #FFB86CA6;">存储空间不连续</mark>
2. 各数据元素的存储顺序和逻辑循序可以不一致；<mark style="background: #FFB86CA6;">存储逻辑不一致</mark>
3. 线性表的链式存储所占存储空间大于顺序存储结构；<mark style="background: #FFB86CA6;">占用空间大</mark>
5. 查找结点时链式储存要比顺序存储慢；<mark style="background: #FFB86CA6;">查找慢</mark>
6. 链式存储插入删除元素比顺序存储灵活；<mark style="background: #FFB86CA6;">插入和修改灵活</mark>
#### 树与二叉树
##### 树
是n（n>0）个元素的有限集合。它有且仅有一个称为根的元素；其余元素是互不相交的子树。
##### 二叉树
是一个有限的结点集合，该集合或者为空，或者有一个根结点及其两颗互不相交的左右二叉子树所组成。

##### 二叉树特点
1. 非空二叉树只有一个根结点，每个结点最多有两颗子树，分别称为左子树和右子树
2. <mark style="background: #BBFABBA6;">在二叉树的第K层上，最多有2k-1个结点；</mark>
3. <mark style="background: #BBFABBA6;">深度为m的二叉树最多有2m-1个结点；</mark>
4. 度为0的结点（叶子结点）总比度为2的结点多一个；
5. <mark style="background: #BBFABBA6;">有n个结点的二叉树深度至少为[log2n]+1。</mark>
##### 二叉树遍历
![[../../_Repertory/计算机二级公共基础_pic/f02c8c337504b3c91186ba6bb34ae7092162c4ee.jpg@!web-article-pic.webp]]
按照一定的顺序访问二叉树中的结点，每个结点只被访问一次
1. 前序遍历：ABDGECF，访问根结点、前序遍历左子树、前序遍历右子树（根左右）
2. 中序遍历：DGBEAFC，中序遍历左子树、访问根结点、中序遍历右子树（左根右）
3. 后序遍历：GDEBFCA，后序遍历左子树、后序遍历右子树、访问根结点（左右根）
